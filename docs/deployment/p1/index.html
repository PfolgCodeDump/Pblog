第一部分：环境准备

安装开发环境配置与项目初始化

第一部分：环境准备

1.1 系统要求

  • Node.js 18.x 或更高版本
  • npm 或 yarn 包管理器
  • Git（推荐用于版本控制）
  • 代码编辑器（如 VS Code、Sublime Text）

1.2 安装 Node.js

Windows

  1. 访问 nodejs.org

  2. 下载 LTS 版本安装包

  3. 运行安装程序，按提示完成安装

  4. 验证安装：

    node --version
npm --version

macOS

使用 Homebrew 安装：

brew install node

Linux (Ubuntu/Debian)

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

1.3 获取项目代码

从 Git 仓库克隆

git clone <repository-url>
cd <project-directory>

或使用现有项目

直接进入项目根目录。

1.4 安装依赖

在项目根目录执行：

npm install

若遇到网络速度较慢，可使用国内镜像：

npm config set registry https://registry.npmmirror.com
npm install

1.5 验证环境

启动开发服务器验证：

npm run dev

访问 http://localhost:3000 确认项目正常运行。

1.6 常用命令

命令 说明
npm run dev 启动开发服务器
npm run build 构建静态网站
npm start 预览构建结果
npm run lint 代码检查
