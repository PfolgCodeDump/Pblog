# Python 简介

Python 是一种高级、解释型、通用的编程语言，由 Guido van Rossum 于 1991 年首次发布。

## 为什么选择 Python

- **简洁易学**: 语法简洁清晰，易于学习
- **应用广泛**: 可用于 Web 开发、数据分析、人工智能等
- **生态丰富**: 拥有大量第三方库和框架
- **跨平台**: 可在多种操作系统上运行
