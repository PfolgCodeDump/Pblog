# C++ 简介

C++ 是一种通用的编程语言，由 Bjarne Stroustrup 于 1979 年在贝尔实验室开始设计开发。

## 为什么选择 C++

- **高性能**: 直接访问硬件，内存管理可控
- **面向对象**: 支持类、继承、多态等特性
- **标准库丰富**: STL 提供了大量数据结构和算法
- **跨平台**: 可在多种操作系统上编译运行
