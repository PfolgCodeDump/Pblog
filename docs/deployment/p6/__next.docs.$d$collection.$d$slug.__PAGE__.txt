3:Ta40,<h2 id="第六部分常见问题">第六部分：常见问题</h2>
<h3 id="61-环境问题">6.1 环境问题</h3>
<p><strong>Q: Node.js 版本不兼容？</strong></p>
<p>A: 升级到 18.x 或更高版本，推荐使用 nvm 管理版本。</p>
<p><strong>Q: npm install 失败？</strong></p>
<p>A:</p>
<ol>
<li>换国内镜像源：<code>npm config set registry https://registry.npmmirror.com</code></li>
<li>删除 <code>node_modules</code> 和 <code>package-lock.json</code> 重新安装</li>
</ol>
<h3 id="62-配置问题">6.2 配置问题</h3>
<p><strong>Q: 环境变量不生效？</strong></p>
<p>A: 修改环境变量后需重启开发服务器，生产环境需重新构建。</p>
<p><strong>Q: Giscus 不显示？</strong></p>
<p>A: 检查：</p>
<ol>
<li>Giscus App 已安装</li>
<li>仓库启用 Discussions</li>
<li>环境变量配置正确</li>
<li>网站已部署到线上</li>
</ol>
<p><strong>Q: Live2D 不显示？</strong></p>
<p>A: 检查模型路径、文件是否存在，或设为空数组禁用。</p>
<h3 id="63-构建问题">6.3 构建问题</h3>
<p><strong>Q: 构建失败？</strong></p>
<p>A:</p>
<ol>
<li>查看错误信息</li>
<li>运行 <code>npm run lint</code> 检查代码</li>
<li>检查导入路径是否正确</li>
</ol>
<p><strong>Q: 图片不显示？</strong></p>
<p>A: 检查：</p>
<ol>
<li><code>NEXT_PUBLIC_SITE_URL</code> 配置正确</li>
<li>使用相对路径引用图片</li>
<li>图片文件已复制到输出目录</li>
</ol>
<h3 id="64-部署问题">6.4 部署问题</h3>
<p><strong>Q: 部署后 404？</strong></p>
<p>A:</p>
<ol>
<li>确认 Output Directory 设为 <code>output</code></li>
<li>检查 <code>next.config.js</code> 中 <code>trailingSlash: true</code></li>
<li>确认链接以 <code>/</code> 结尾</li>
</ol>
<p><strong>Q: 样式丢失？</strong></p>
<p>A: 检查 <code>NEXT_PUBLIC_SITE_URL</code> 配置正确，强制刷新浏览器。</p>
<h3 id="65-性能问题">6.5 性能问题</h3>
<p><strong>Q: 网站加载慢？</strong></p>
<p>A:</p>
<ol>
<li>优化图片大小和格式</li>
<li>禁用 Live2D（如使用）</li>
<li>减少粒子数量</li>
<li>使用 CDN 加速</li>
</ol>
<h3 id="66-其他问题">6.6 其他问题</h3>
